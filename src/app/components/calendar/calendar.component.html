Manage this slot

This action creates one or more 'available' slots, which Lifecoach users can use to schedule free discovery video calls with you. Think of each available slot as a bookable calendar slot which can be used by a single person to schedule a video call with you. Set a short start and end time to create a single slot or set a longer time period to automatically create multiple available slots. Visit 'Account > Session Settings' to manage your default slot times. Slots can easily be deleted at any time before they are booked.
